Reading & Learning

Here are some of the resources I’ve found especially valuable on my journey as a software engineer:

  • System Design Interview – Alex Xu A clear and practical introduction to system design. It covers core components like CDNs, load balancers, and databases in a structured way — a great starting point for preparing for system design interviews.

  • Algorithms I + II – Princeton University (Coursera) An excellent, university-level course that revisits data structures and algorithms with real-world examples and coding assignments. I highly recommend it as a foundation before diving into technical interview prep.

  • Python Django Full Course for Beginners – Dave Gray (YouTube) A fast-paced, beginner-friendly course that helped me get up to speed with Django. It’s clear, hands-on, and perfect for building your first Django project.
